# Laravel Old Helpers

## Installation

You can install the package via composer:

```bash
composer require takuya829/laravel-old-helpers
```

## Usage

```php
$oldStr = old_str('input_name', 'default value');
echo $oldStr;
```

### Helper functions
- old_str
- old_num
- old_int
- old_float
- old_bool
- old_array

## Testing

```bash
composer test
```
